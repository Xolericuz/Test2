# XOLERIC NEO (Test2)

A sleek book reader app with 50 books, zoom controls, pull-to-close gesture, and full-screen reading mode.

## Features
- 50 books with search
- Font size zoom panel (slider)
- Pull-to-close reader gesture
- Full-screen mode on entry
- Last-read book tracking
- Blur-mode security on window blur
- Keyboard shortcut blocking

## How to open
Open `index.html` in any modern browser.
